# EMS-Compatible Package

This package is generated directly from Copernicus EMS GRA/BLP vectors.

- Input layer: `builtUpA_v1`
- Key EMS field: `damage_gra`
- Outputs: CSV, GeoJSON, KML, static map viewer

For AOI12 La Guaira, run this same builder on the AOI12 GRA zip when it becomes downloadable.
